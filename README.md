# MEVN stack

mysql + express + vuejs + nodejs stack

It's a very simple starter kit of MEVN stack to get me started with simple JS projects.

# Technologies used:-

- Vuejs for frontend.
- NodeJs for backend.
- ExpressJs for server-side/API requests
- Mysql database.
- Sequalize.
- Passportjs.
- Helmet.
- express-validator.
- Winston.
- Vue-notifications.
- vuejs-i18n.
- VeeValidate
